# 📝 Adventure Game - Step-by-Step Instructions

Follow these steps to set up and play the text-based Adventure Game.

---

## ✅ Step 1: Create the Python File

- Open any folder where you want to save the game.
- Create a new file named `adventure.py`.

---

## ✅ Step 2: Copy and Paste the Code

- Copy the game code into `adventure.py`.
- Save the file.

---

## ✅ Step 3: Open Command Prompt

- Navigate to the folder where `adventure.py` is located using `cd` command.

Example:
```cmd
cd C:\Users\YourName\Downloads\game_folder
```

---

## ✅ Step 4: (Optional) Create Virtual Environment

If you want to use a virtual environment:

```cmd
python -m venv env
env\Scripts\activate
```

> This game does not require any external libraries, so this step is optional.

---

## ✅ Step 5: Run the Game

Type the following command in your terminal:

```cmd
python adventure.py
```

---

## ✅ Step 6: Play the Game

- Follow the prompts in the terminal.
- Make your choices by typing the options.
- Each decision will lead you to a different outcome.

---

## ✅ Example

```plaintext
Type your name: Swapnil
Welcome Swapnil to this adventure
You are on a dirt road, it has come to an end and you go left or right. Which way would you like to go? left
You come to a river, you can walk around it or swim across? Type walk to walk around and swim across: walk
You walked for miles, ran out of water and you lost the game
Thank you for trying Swapnil
```

---

Enjoy the game! 🎮

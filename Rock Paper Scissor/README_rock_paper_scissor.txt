# 🎮 Rock, Paper, Scissors – Terminal Edition

An upgraded version of the classic Rock, Paper, Scissors game built in Python, enhanced with terminal colors, emojis, ASCII-like art, and smooth effects — no external libraries required!

---

## 🧠 Features

- 🖼️ Visually appealing terminal output using ANSI colors and emojis  
- 🎨 Rock 🪨, Paper 📄, Scissors ✂️ visual indicators  
- 🔁 Continuous play loop with a clean exit  
- 📊 Real-time score tracking  
- ❌ Input validation and instant feedback  
- 👋 Friendly game intro and outro with animated effects  

---

## ▶️ How to Run

1. Save the code in a file called `rock_paper_scissors_terminal.py`.  
2. Open terminal or command prompt.  
3. Navigate to the script's folder.  
4. Run the game with:

```bash
python rock_paper_scissors_terminal.py

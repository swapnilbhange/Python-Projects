# 🎨 Terminal Edition: Advanced Number Guessing Game

A visually engaging terminal-based number guessing game with colored text, ASCII effects, and smooth animations — all using pure Python!

---

## 💡 Features

- 🔢 User-defined lower and upper bounds  
- 🧠 Difficulty Levels (Easy, Medium, Hard)  
- 🎨 Color-coded feedback:
  - Green = Success
  - Yellow = Warning / Close
  - Red = Incorrect  
- 🔥 Hints like "Very close", "Too high", "Too low"  
- 🏆 ASCII Art for WIN / GAME OVER  
- ⌛ Smooth "typewriter-style" print effects  
- 🔁 Replay option after the game ends  

---

## 🚀 How to Run

1. Save the script as `advanced_guessing_game_colored.py`.
2. Open terminal/command prompt and navigate to the file location.
3. Run the game:

```bash
python advanced_guessing_game_colored.py
